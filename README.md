# ImHex Protobuf Test Case

Just a sample protocol buffer to use as a test case for ImHex protobuf parsing.

* [small.png](small.png): a very small PNG file to embed in a proto.
* [testcase.pb](testcase.pb): the test proto in binary wire format.
* [testcase.proto](testcase.proto): the proto message definition used in the test case.
* [testcase.protoscope](testcase.protoscope): the test proto in text format - output from parsing with https://github.com/protocolbuffers/protoscope.
* [testcase.txtpb](testcase.txtpb): the "official" text format of the test proto.
